chapter 2. 통계란

통계의 기본 개념과 용어

  1. 모집단과 표본
    • 모집단 : 관심있는 문제와 연관된 전체 집합
    • 표본 : 전체 데이터로부터 추출된 일부 집합
  2. 파라미터와 통계량
    • 파라미터 : 모집단으로부터 얻은 데이터의 성격을 나타내주는 지표(모평균, 모분산 등)
    • 통계량 : 표본으로부터 얻은 데이터의 성격을 나타내주는 지표(표본 평균, 표본 분산 등)
  3. 표집오차(sampling error)
    • 모집단과 표본의 차이로 발생하는 오차
    • 아무리 많은 노력을 기울여 지능적인 표집 방법을 사용한다 하더라도 표본과 모집단의 특성이 100% 일치할 수는 없음
    • 모집단이 아닌 표본을 사용한다면 언제나 표집오차가 존재하므로 이를 염두에 두고 데이터를 사용해야 함
    • 결국 머신러닝에 사용하는 모든 학습용 데이터셋이 과연 얼마나 모집단을 잘 반영하는지에 대해 검증해 보아야 한다는 말
  4. 종속변수와 독립변수
    • 종속변수
      • 어떤 목적에 대한 결과를 파악하기 위해 사용하는 변수
      • 반응변수(response), 측정변수(measured), 피예측변수(predicted), 피설명변수(explained), 출력변수(output) 등으로 불림
    • 독립변수
      • 예측변수(predictor), 특징(feature), 입력변수(input), 설명변수(explanatory) 등으로 불림
  5. 연속변수와 이산변수
    • 통계처리를 하려는 값의 특성에 따라 분류한 것
    • 연속변수
      • 실수로 표현할 수 있는 변수
      • 특정 범위에서 허용되는 값이 '셀 수 없다'
    • 이산변수
      • '셀 수 있는' 변수
    • 이를 구분해야 하는 이유는 이에 따라 적절한 방법을 선택해야 하기 때문
  6. 모델
    • 대상을 설명하기 위한 표현 방법
